中润检测项目开发说明
  • 一、检测项目开发说明
    • 1. Helper 类继承接口 ICHK
    • 定义一个常量： private const string ZrCode = "A0054";
    • 显式实现一个方法 ICHK.GetZrCode() return ZrCode
    • 2. Trust （收样窗体）继承 Trust
    • 覆写 Base 的 InitBaseCheckClassInfo 方法，并且调用
    • 3. Trial （试验窗体）继承 Trial
  • 二、检测项目基类说明
    • 1. Base 类继承接口 IDragControl ，显式实现接口 IDragControl 的所有方法
    • 2. Trust （收样窗体的父类）继承 Base, ICHK_Trust ，隐式实现接口 ICHK_Trust 的所有方法
    • 覆写 Base 的 InitBaseCheckClassInfo 方法，并且调用
    • 3. Trial （试验窗体的父类）继承 Base, ICHK_Trial ，隐式实现接口 ICHK_Trial 的所有方法
  • 三、调用说明
    • 1. App 装载 DLL 程序集，判断是否含有接口 ICHK ，否则 return
    • 2. 调用接口方法 ICHK.GetZRCode() 判断 ZrCode 是否相同，否则 return
    • 3. 根据业务调用具体接口
      • 收样调用接口 ICHK_Trust
      • 试验调用接口 ICHK_Trial
      • 拖拽调用接口 IDragControl
      • 注意：接口调用不要用匿名委托
  • 四、窗体设计需求文档规范说明
    • 1. 第一页作为备用说明页，不做任何读取。收样内容放在第二页，试验内容放在第三四五...页。
    • 2. 退出标志为 红色填充的单元格（第一列某一行）（最多也就读取 1000 行）
    • 3. 收样就一页
    • 4. 试验是选项卡，如果内嵌选项卡需要严格按照格式（暂时没有实现其他容器控件的内嵌，如 Panel 与 GroupBox,及其他容器控件）
    • 4.1 标识为 tabcontrol 的下一行，只能填 tabpage 信息，且合并列（读取占用宽度）
    • 4.2 上点所说的所占用行数必须一样，即所有的都取第一个的所占用行数
  • 五、数据库字段对照需求文档规范说明
    • 1. 第一页作为备用说明页，不做任何读取。字段内容放在第二三四...页。
    • 2. 退出标志同窗体设计文档
